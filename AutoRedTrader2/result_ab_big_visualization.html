Same adversarial attack tape replayed against the plain agent and the grounded (defended) agent — isolates the effect of the defense itself.

model=qwen3:14b · asset=BTC-USD · days=10 · seeds=3 · strict_judge=True · τ_det=0.6

The grounding defense did not reduce attack success — it increased flips by 3.3 points (Attack Success Rate: 16.7% plain → 20.0% grounded), while return on injected days (CR_inj) dropped from +3.4% to +1.8%. MER is ~equal across agents, confirming the attack itself was identical — the gap is the defense's own effect.

How to read this

  • ASR is the defense metric. Plain 16.7% vs grounded 20.0% → the defense increased flips by 3.3 points; it did not help.
  • MER is ~equal across agents (same injected items in both replays) — confirming this is a fair, identical-attack comparison; any residual gap is retrieval tie-breaking.
  • CR is a side effect, not the defense metric — grounding can lower realized return (more caution) while still failing to cut ASR.
